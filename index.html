Resources

Category

  • Github for Mac app, or Github for Windows app: this is a regular app (just like Chrome or Microsoft Word) - no command line needed! You can use this app to practice your Git workflow without having to remember the commands. While most developers use the command line because it is more powerful, the GUI (Graphic User Interface) is a great way to get started!
  • Try Git: an easy to use online interface for learning Git commands
  • Git Basics: a free video tutorial that covers the basics of Git
  • The Simple Guide to Git: a step-by-step guide to using Git with explanations for each command
  • Learn Git Branching: a more advanced topic, this covers branching and allows you to walk through every command.
